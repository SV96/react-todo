# App name : Todo

# Technologies used

> React
> Redux
> material UI
> moment

**It's a web app created in React Js. Here user can added,edit and delete todo.**

## Todo fields

> User name, Gender , Hobby, Age, Date, Task, Status, Action

## To run project

    git clone https://github.com/SV96/react-todo.git
    cd react-todo
    npm install
